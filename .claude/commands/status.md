Show the current status of the Belva-GEN project.

Usage: /status

Run these commands and report the output:

1. `git status` — working tree status
2. `git log --oneline -5` — recent commits
3. `npm run quality` — lint + type-check status
4. `ls .claude/agent-memory/*/MEMORY.md` — agent memory files
5. `ls .github/skills/*/SKILL.md` — available skills

Summarize the project health: branch, uncommitted changes, quality status, and available governance artifacts.
